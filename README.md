# job-application-form
A clean, structured job application form built with HTML and CSS. Designed to collect applicant information including personal details, job preference, address, and CV upload in a professional layout.

## Live Demo
View Live 🌐
https://ankamahjohnson.github.io/job-application-form/


## Features
- Clean minimal form layout on a light grey background
- Required field validation indicators with red asterisks
- First name and last name input fields
- Email input with placeholder formatting
- Job role dropdown selector
- Address textarea for multiline input
- City and Pincode input fields
- Date selector field
- CV upload input with file chooser
- Orange accent border styling on input focus
- Bold red "Apply Now" submit button
- Fully responsive layout
